# Palavras Proibidas - Doma Pro

> Palavras e expressões que enfraquecem a comunicação e afastam o público-alvo.

---

## 1. Jargões de Marketing Digital

| Evitar | Por quê |
|--------|---------|
| "Transforme sua vida" | Clichê batido, público rural não se conecta |
| "Desbloqueie seu potencial" | Linguagem de coach, não de domador |
| "Mindset" | Estrangeirismo que soa artificial |
| "Método revolucionário" | Promessa exagerada, gera desconfiança |
| "Exclusivo" (em excesso) | Perde força quando repetido |
| "Incrível" / "Fantástico" | Adjetivos vazios |
| "Oportunidade única" | Clichê de vendedor |
| "Não perca essa chance" | Pressão artificial |
| "Vagas limitadas" (falso) | Se for mentira, destrói credibilidade |
| "Fórmula secreta" | Soa como charlatanismo |
| "Hackeie" / "Hack" | Linguagem de startup, não de campo |

## 7. Termos que Afastam o Público Rural

| Evitar | Por quê |
|--------|---------|
| "Premium" | Soa elitista |
| "High-end" | Estrangeirismo desnecessário |
| "Upgrade" | Usar "melhoria" ou "evolução" |
| "Top" (em excesso) | Perde força |
| "Pro" sem contexto | Já está no nome, não exagerar |
| "VIP" | Distante da realidade do campo |
| "Elite" | Pode soar excludente |
| "Sofisticado" | Não é o que o público busca |

## 9. Palavras com Conotação Negativa no Contexto

| Evitar | Por quê |
|--------|---------|
| "Barato" | Use "acessível" ou "investimento justo" |
| "Gasto" | Use "investimento" |
| "Comprar" (em excesso) | Use "entrar", "fazer parte", "assinar" |
| "Pagar" (em excesso) | Use "investir" |
| "Custo" | Use "valor" ou "investimento" |
| "Problema" (sobre o cliente) | Use "desafio" ou "situação" |
| "Difícil" | Use "desafiador" |
| "Amador" | Pode ofender, use "iniciante" |

## 10. Expressões que Quebram Confiança

| Evitar | Por quê |
|--------|---------|
| "Confie em mim" | Se precisa pedir, já perdeu |
| "Acredite" | Mesma coisa |
| "Eu prometo" | Promessas verbais são fracas |
| "Palavra de honra" | Antiquado e vazio |
| "Pode ter certeza" | Insegurança disfarçada |
| "Sem pegadinha" | Levanta suspeita de pegadinha |
| "Isso não é golpe" | Nunca diga isso |

---

## Regras Gerais

1. **Fale como o André fala** - "Fala primos" funciona. "Prezado cliente" não.
2. **Seja direto** - Menos adjetivos, mais fatos.
3. **Use números** - "2.500 alunos" é melhor que "milhares".
4. **Mostre, não diga** - Depoimentos > promessas.
5. **Respeite a inteligência** - Público rural não é ingênuo.
6. **Evite estrangeirismos** - Se existe palavra em português, use.
7. **Cuidado com superlativos** - "Melhor", "único", "maior" precisam de prova.
