# V10 · Negative Reverse · Esse Anúncio Não é Pra Todo Mundo

**Framework:** Negative Reverse / Disqualify-to-Qualify
**Duração:** ~30s
**Público:** Retargeting · qualificação
**Gancho:** "Doma Pro não é pra todo mundo."
**Por que funciona:** efeito psicológico de reverso. Quando você diz que não é pra todo mundo, quem se identifica passa a querer mais. Filtra cliente errado e atrai o certo.
**Estratégia:** anúncio → clique no CTA → Landing Page → checkout
